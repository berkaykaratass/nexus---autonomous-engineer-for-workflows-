## Overview

Nexus is an advanced, autonomous AI engineering platform designed to handle complex software workflows, automate tasks, and streamline the development lifecycle. This repository showcases the frontend interface and core interaction modules for the Nexus platform.

## Features (UI Preview)

- **Interactive AI Developer Sandbox:** Chat with the Nexus AI engineer and watch it write, test, and deploy code.
- **Job Orchestration:** View automated jobs, cron triggers, and event-based workflows.
- **Codebase Integration:** Connect to Git repositories and see AI-driven code diffs live.
- **Terminal Simulation:** Live execution logs of the AI's background processes.

## Tech Stack (Frontend)

- React 19 / TypeScript
- Vite
- Tailwind CSS
- Recharts (for Analytics)
- Lucide React (Icons)

## Getting Started

To run this UI locally:

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```
